Getting Started

Brew Brain is a fermentation monitoring and automation system designed for the Grainfather G40 with SS Brewtech Unitanks.

First-Time Setup

  1. Connect your Tilt Hydrometer to TiltPi
  2. Configure Brewfather API in Settings
  3. Set up Telegram Bot for alerts
  4. Enter your Batch Details (OG, Target FG)

Dashboard

The main dashboard shows real-time fermentation data:

Temperature

Current fermenter temp from Tilt

Gravity (SG)

Specific gravity reading

Attenuation

% of sugars consumed

ABV

Estimated alcohol content

Settings & Calibration

Batch Settings

  • Batch Name: Identifier for current brew
  • Start Date: When you pitched yeast
  • OG: Original Gravity (e.g., 1.055)
  • Target FG: Expected finish (e.g., 1.010)

Tilt Calibration

If Tilt reads different from your hydrometer:

  1. Take a sample and read with manual hydrometer
  2. Enter the actual reading in Calibration box
  3. Click Set

The offset is calculated automatically.

Brewfather Integration

Sync recipe data, yeast specs, and inventory from Brewfather.

Setup

  1. Go to Brewfather ‚Üí Settings ‚Üí API & Power Controller
  2. Enable Custom Stream and copy your User ID
  3. Generate an API Key
  4. Enter both in Brew Brain Settings
Your batch must be set to "Fermenting" status in Brewfather to sync.

Telegram Alerts

1. Create a Bot

Talk to @BotFather on Telegram. Send /newbot and follow instructions.

2. Get Your Chat ID

Talk to @userinfobot to get your numeric ID.

3. Enter in Settings

Paste both Token and Chat ID into Brew Brain.

Bot Commands

/status Get current readings
/ping Check connection
/prediction Get FG prediction

ML Predictions

Brew Brain uses Physics-Informed Machine Learning to predict fermentation outcomes.

4PL Sigmoid Model

A 4-Parameter Logistic curve is fitted to your gravity readings to predict:

  • Final Gravity (FG) - Where fermentation will stop
  • Time to Terminal - Hours until completion
  • Confidence Score - R¬≤ fit quality (High/Moderate/Low)

Yeast Memory

The system learns from your brewing history:

  • Historical attenuation rates per yeast strain
  • Expected fermentation velocity
  • Anomaly detection (infection warning)

Troubleshooting

No data showing?

Check Tilt is in range, TiltPi is running, and both are on same network.

Brewfather sync failing?

Verify API credentials and ensure batch is set to "Fermenting" status.

Telegram not working?

Start a chat with your bot first. It can't message you until you message it.

Price comparison shows N/A?

Check SerpApi key is valid. Some specialty malts may need manual lookup.

User Guide

Batch Settings

Configure these in the Settings tab to ensure accurate stats.

  • Start Date: When you pitched the yeast.
  • OG (Original Gravity): Measured before fermentation (e.g., 1.050).
  • Target FG: Expected finishing gravity (e.g., 1.010).

Setup Alerts

1. Create a Bot

Talk to @BotFather on Telegram. Send /newbot and follow instructions to get your Token.

2. Get Chat ID

Talk to @userinfobot on Telegram to get your numeric ID.

3. Connect

Enter the Token and Chat ID in Brew-Brain Settings.

Bot Commands

/status Get current readings
/ping Check connection

Calibration

If your Tilt reads 1.020 but your manual hydrometer says 1.015:

  1. Go to Settings > Calibration.
  2. Enter 1.015 in the box.
  3. Click Set.

The system calculates the offset (-0.005) automatically.

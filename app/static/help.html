User Guide

Batch Settings

Configure these in the Settings tab to ensure accurate stats.

  • Start Date: When you pitched the yeast.
  • OG (Original Gravity): Measured before fermentation (e.g., 1.050).
  • Target FG: Expected finishing gravity (e.g., 1.010).

Calibration

If your Tilt reads 1.020 but your manual hydrometer says 1.015:

  1. Go to Settings > Calibration.
  2. Enter 1.015 in the box.
  3. Click Set.

The system calculates the offset (-0.005) automatically.

AI & Intelligence

The Brain uses Physics-Informed ML and Yeast Memory to protect your brew.

  • Sync: Downloads Yeast Specs (Temp Range, Attenuation) from Brewfather.
  • Safety: Alerts if you exceed the Manufacturer's Max Temp.
  • Anomaly: Warns if fermentation is suspiciously fast (>2.5x normal rate).

Brewfather

Sync recipe data (Target F.G, OG, Name) from Brewfather.

  1. Go to Brewfather Settings > Power Controller & API.
  2. Generate an API Key.
  3. Enter User ID and Key in Brew-Brain.
  4. Click Save, then Sync Batch.
Must have a batch status set to "Fermenting" in Brewfather.
